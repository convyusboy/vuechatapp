# vueappchat

## Recommended IDE Setup

[VSCode](https://code.visualstudio.com/) 



## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Run the App Server

open new terminal

```sh
cd vuechatapp
```
```sh
cd server
```
```sh
node index.js
```

the chat app will be on http://localhost:5173/
the chat app server will be on http://localhost:5001/
